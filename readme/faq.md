# ❓ FAQ

### How to automatically open private messages in a new tab? <a href="#how_to_automatically_open_private_messages_in_a_new_tab" id="how_to_automatically_open_private_messages_in_a_new_tab"></a>

Go to Options → Console and _deselect/uncheck_ “Show private messages in console”. This will make private messages open up a new tab.

![](https://wiki.soerpg.com/lib/exe/fetch.php?w=300\&tok=226ef4\&media=faq:private_messages_new_tab.png)

### Is there an in-game market? <a href="#is_there_an_in-game_market" id="is_there_an_in-game_market"></a>

No. Souls of Elysium promotes player to player interaction. You can use the Advertising channel to sell and buy items.

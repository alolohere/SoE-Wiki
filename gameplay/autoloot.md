# Autoloot

### Autoloot on Souls of Elysium <a href="#autoloot_on_souls_of_elysium" id="autoloot_on_souls_of_elysium"></a>

* **Souls of Elysium** offers a built in Autoloot feature!
* You'll be able to interact with the system via the icon below.

![](<../.gitbook/assets/image (186).png>)

* Here you can search for items to add, or manually drop items into the Autoloot slots.

<div align="left"><figure><img src="../.gitbook/assets/image (187).png" alt="" width="249"><figcaption></figcaption></figure></div>



#### Expanding the Autolooter <a href="#expanding_the_autolooter" id="expanding_the_autolooter"></a>

* The Autolooter can be expanded with additional slots and functionality.
* The tokens for expanding the autolooter are available in the client [Game Store](../cosmetics/store.md) in the Extra Services category.

| Item Name                | Picture                                                                                                    | Cost from Store | Notes                                                                                                                                                                                                                    |
| ------------------------ | ---------------------------------------------------------------------------------------------------------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Autoloot Slot Token      | <div><figure><img src="../.gitbook/assets/image (188).png" alt=""><figcaption></figcaption></figure></div> | 50              | You can increase the 5 autoloot slots that are available from the start up to a total of 20.                                                                                                                             |
| Autoloot Filter Token    | <div><figure><img src="../.gitbook/assets/image (189).png" alt=""><figcaption></figcaption></figure></div> | 250             | This allows you to automatically loot all items and instead select which items to ignore.                                                                                                                                |
| Autoloot Preset Token    | <div><figure><img src="../.gitbook/assets/image (190).png" alt=""><figcaption></figcaption></figure></div> | 150             | At the top of the Autoloot window you can switch between presets. You can have a total of 10 different presets.                                                                                                          |
| Autoloot Container Token | <div><figure><img src="../.gitbook/assets/image (469).png" alt=""><figcaption></figcaption></figure></div> | 100             | There are two unlockable containers, _Stackables_ and _Useables_. This will automatically put items like Coins and Small Ruby into the backpack selected for _Stackables_ and potions etc. into the _Useables_ backpack. |

# Mining

## Mining on Souls of Elysium <a href="#mining_on_souls_of_elysium" id="mining_on_souls_of_elysium"></a>

**Souls of Elysium** provides a new and interactive mining system allowing players to interact with ore nodes in the world, and craft powerful items!

### Mining Pick <a href="#mining_pick" id="mining_pick"></a>

* Obtained from any tool shop for 1500 gold pieces, or by completing [Gilion's Quest](gilions-quests.md) on top of a mountain in Kazordoon (farms area).
* You can take the Magic Carpet from Darashia to get dropped extremely close.
* Simply right click the pickaxe, and left click the first copper ore you come across to use it.
* _Note: Once you've completed Gilion's Quest, you'll be able to directly right click on the ores to mine them._

### Where do I mine? <a href="#where_do_i_mine" id="where_do_i_mine"></a>

* Ore rocks are most commonly found out in the world.
* The higher tier the ore, the more dangerous a place it may be hiding.
* Copper ore can be found in plenty of areas accessible to New Players.
* Minotaur Tower
* Tombs in darashia/ankrahmun desert
* Femur Hills
* Many other places!

<figure><img src="../../.gitbook/assets/image (164).png" alt=""><figcaption></figcaption></figure>



* In addition to the veins you can find out in the world, there are infinite ore mines.
* Progression in the mining skill is slower, but the nodes can be mined infinitely.
* There are currently three mines, located in Kazordoon, Hellgate and Narnia.
* [Kazordoon Mine](kazordoon-mine.md) (Coper, Iron ores)
* [Hellgate Mine](hellgate-mine.md) (Coal, Gold and Mithril ores)
* [Narnia Mine](hellgate-mine.md) (Coal, Adamant and Runite ores)

### Upgrading Pickaxe <a href="#upgrading_pickaxe" id="upgrading_pickaxe"></a>

* Your pickaxe can be upgraded at a crafting station.
* Each upgrade will increase the number of times a node is mined before having to click again.
* Increased chance of receiving more than one ore.

### Mining Events <a href="#mining_events" id="mining_events"></a>

* For every hit on an ore vein there is a chance to receive extra rewards or spawn a monster.

| Multiplier        | Description                                                                      |
| ----------------- | -------------------------------------------------------------------------------- |
| `vein_difficulty` | The mining level required to mine the ore vein. Copper 10, Iron 20, Coal 25 etc. |
| `pickaxe_tier`    | The tier of the pickaxe. Primitive is 1, Rune is 7.                              |
| `mining_skill`    | The mining skill level the player has.                                           |
|                   |                                                                                  |

| Item           | Formula                                                                      |
| -------------- | ---------------------------------------------------------------------------- |
| Old Black Box  | `1 / (1500 - (10 * vein_difficulty + 40 * pickaxe_tier + 3 * mining_skill))` |
| Quartz         | `2 / (1500 - (10 * vein_difficulty + 40 * pickaxe_tier + 3 * mining_skill))` |
| Crystal Fossil | `6 / (1500 - (10 * vein_difficulty + 40 * pickaxe_tier + 3 * mining_skill))` |
| Monster        | `7 / (1500 - (10 * vein_difficulty + 40 * pickaxe_tier + 3 * mining_skill))` |
|                |                                                                              |

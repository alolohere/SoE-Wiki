# Leveling Guides

### Suggested Tasks <a href="#suggested_tasks" id="suggested_tasks"></a>

* Common enemy tasks (Spiders, Wolves, Snakes) are good for early game exp and money.



#### Knight <a href="#knight" id="knight"></a>

| Area/Monsters                                       | Level | Extra Info                                                                                          |
| --------------------------------------------------- | ----- | --------------------------------------------------------------------------------------------------- |
| Trolls                                              | 8+    | Choose your favorite spawn.                                                                         |
| Orcs                                                | 8+    | Under Thais or at Orc Fortress                                                                      |
| Dwarves                                             | 8+    | Under Kaz                                                                                           |
| Minotaurs                                           | 10+   | Minotaur Tower, be careful of Mages/Guards                                                          |
| Larva/Scarabs                                       | 10+   | Darashia/Ank Tombs.                                                                                 |
| Wild Warriors                                       | 20+   | Darashia Arena (Take Gladiator task too) or POH Camp.                                               |
| Valkyries                                           | 20+   | Venore (Take Amazon task too)                                                                       |
| Barbarians                                          | 30+   | Svargrond                                                                                           |
| Cyclops                                             | 30+   | Edron location or Mount Sternum, be careful of Diabolics/Demonics.                                  |
| Ember Beetles                                       | 40+   | Darashia or Thais location.                                                                         |
| Ancient Scarabs                                     | 50+   | Darashia/Ank Tombs. Take Bonebeast/Scarab/Larva Tasks.                                              |
| Dragons                                             | 50+   | Darashia or Venore Dragon Lair.                                                                     |
| Giant Spiders                                       | 60+   | Requires Goroma Access for [Custom Spawn](../exploration/custom-spawn.md).                          |
| Dragons/Dragon Lords                                | 70+   | Dragons are easy, be careful of Dragon Lords if they spawn a Diabolic/Demonic.                      |
| Heroes                                              | 100+  | Edron Spawn, Can easily overwhelm low leveled players, Diabolics/Demonics can be dangerous as well. |
| Vicious Squires, Renegade Knights, Vile Grandmaster | 100+  | [Custom Spawn](../exploration/custom-spawn.md)                                                      |
| Giant Cobras                                        | 100+  | Port Hope has 2 spawns                                                                              |
| Hydras                                              | 100+  | Goroma or Port Hope.                                                                                |
| Frost Dragons                                       | 100+  | Okolnir, Chyllfrost, can drop Avalanche Rune Scroll.                                                |
| Fiery Beetles                                       | 100+  | Darashia or Thais location.                                                                         |

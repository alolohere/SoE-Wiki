# Weekly Events

## Events

* In Souls of Elysium there are weekly events for a chance at rewards!
* These can vary from raids on cities to events you join via command.



### [Neptulos](neptulos.md)

* A weekly Boss event, taking place on Saturday at 1 PM EST (5PM GMT)
* Safe death!



### Dream Haunter

* A weekly Boss raid, taking place on Sundays at X
* NOT safe!
